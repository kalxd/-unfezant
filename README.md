# 这是什么

基于[rumqtt](https://github.com/bytebeamio/rumqtt)的一个MQTT v3版本的GUI，
用于观察事件的收发状态。

启动应用时，自身自带一个broker和一个client，订阅于`hello/world`，之后设备间的通讯，都会在日志显示。
